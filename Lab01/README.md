# Lab01 — Stringhe, Ordinamento e Multiordinamento

### Es1 — Espressioni regolari
Funzione `char *cercaRegexp(char *src, char *regexp)` che trova la prima occorrenza
di una regexp in una stringa. Metacaratteri: `.` `[abc]` `[^abc]` `\a` `\A`

### Es2 — Azienda di trasporti
Menu con `enum comando_e` per interrogare un log di corse letto da `corse.txt`.
Comandi: `date`, `partenza`, `capolinea`, `ritardo`, `ritardo_tot`, `fine`

### Es3 — Ordinamento
Stampa su video/file, ordinamento per data/ora/tratta/partenza/arrivo,
ricerca lineare e dicotomica (anche per prefisso).

### Es4 — Multiordinamento
Più ordinamenti simultanei con vettori di puntatori a struct,
senza modificare il vettore originale.
